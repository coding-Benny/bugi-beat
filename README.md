# Bugi Beat
> 네트워크 리듬게임, Bugi Beat🎶
## 🎈 Features
- 채팅
  - 입퇴장 알림
  - 자신이 작성한 메시지는 별도의 색상으로 표시
  - 이모티콘
- 게임
  - 방 생성
  - 곡 선택
  - 게임 준비 / 시작
  - 상대 플레이어의 닉네임, 점수, 화면 표시
  - 방해 아이템
- 랭킹
- 접속자 명단
